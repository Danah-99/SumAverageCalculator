# SumAverageCalculator

A simple Java program that:
- Takes 3 numbers from the user
- Calculates the sum using a method
- Calculates the average using another method
- Displays the results

## Technologies Used
- Java
- Scanner (java.util)

## How it works
1. User enters 3 numbers
2. Program calculates sum
3. Program calculates average
4. Results are printed on screen
